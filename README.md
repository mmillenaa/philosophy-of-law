# Philosophy of Law - Interactive Mock Test 📚⚖️

A lightweight, single-file HTML/JS interactive mock test covering classic political and legal philosophy. Developed to support students of the "Philosophy of Law I" course (DFD0313). 

## 🧠 Topics Covered
This quiz consists of 60 rigorous questions based on the theoretical frameworks of:
* **Plato** (The Republic, Book I)
* **Aristotle** (Nicomachean Ethics, Book V)
* **Thomas Hobbes** (Leviathan)
* **John Locke** (Second Treatise of Government)
* **Immanuel Kant** (Categorical Imperative & Perpetual Peace)
* **Jeremy Bentham** (Utilitarianism)

## 🚀 Features
- **Zero dependencies:** Works entirely offline. Just download and open the `.html` file in any modern browser.
- **Real-time Feedback**
- **Dynamic Shuffling:** Alternatives (A, B, C, D) are randomly shuffled on every render to prevent visual memorization.
- **Comprehensive Explanations:** Every question provides an immediate theoretical comment upon answering.

## ⚖️ Disclaimer
This project is an independent academic study tool. The questions and explanations are freely paraphrased from study notes and summaries. It has no official affiliation with the university department or the professors of the course. Use it as a supplementary study aid at your own risk.
